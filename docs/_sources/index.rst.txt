Career Readiness
================

This is a collection of resources for individuals with a solid technical skill set that are in the final stages of preparing for their first tech job. This includes LC101 and CoderGirl graduates, general LaunchCode candidates, or anyone wishing to learn about how to make sure you're ready for that first tech job.
